================================================================================
  TROUBLESHOOTING
================================================================================

‚ùì Build gagal - "PyInstaller not found"
   Solusi: pip install pyinstaller

‚ùì Build gagal - "pywebview not found"  
   Solusi: pip install pywebview

‚ùì .exe tidak jalan - "Missing DLL"
   Solusi: Install Visual C++ Redistributable
   Link: https://aka.ms/vs/17/release/vc_redist.x64.exe

‚ùì .exe tidak bisa connect database
   Solusi:
   - Pastikan ada koneksi internet
   - Cek firewall tidak block aplikasi
   - Test connection string di .env sebelum build

‚ùì Mau rebuild dengan credentials baru
   Solusi:
   1. Edit file .env
   2. Jalankan lagi: python build_exe.py
